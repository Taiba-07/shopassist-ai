# 🛍️ ShopAssist AI

AI-powered e-commerce support assistant using RAG, LangGraph, and Groq.

## 🌐 Live Demo
https://shopassist-ai-rwqsmnejuabgugpf5x4wm7.streamlit.app/

## Features
- 📚 FAQ answering using RAG (ChromaDB)
- 📦 Order tracking simulation
- 🧠 Session memory
- 💬 Chat UI with Streamlit

## Tech Stack
- LangChain / LangGraph
- ChromaDB
- Groq (LLaMA)
- Streamlit

## Run Locally
pip install -r requirements.txt  
streamlit run shopassist/app.py
